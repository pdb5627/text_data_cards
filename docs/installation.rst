============
Installation
============


Stable release
--------------

To install Text Data Cards, run this command in your terminal:

.. code-block:: console

    $ pip install text_data_cards

This is the preferred method to install Text Data Cards, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Text Data Cards can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/pdb5627/text_data_cards

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/pdb5627/text_data_cards/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/pdb5627/text_data_cards
.. _tarball: https://github.com/pdb5627/text_data_cards/tarball/master
